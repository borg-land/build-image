## Hello

This is Upo's CI Build image based on Alpine with Hashicorp Tools and Google SDK.
## How to use the Image:

`docker pull quay.io/upodroid/build-image`

### Applications:

- gcloud
- terraform
- vault
- packer
- kubectl

Images are tagged based on Terraform Version.
